A full-stack Machine Learning project that predicts the sale price of houses using a tuned XGBoost Regressor. 
The app features:
- Real-time prediction from user inputs
- CSV/XLSX batch predictions
- Automated preprocessing pipeline (OneHotEncoding + feature ordering)
- Feature mismatch-proof inference using saved encoders and column order
- Interactive Streamlit web interface
- Fully deployed on Streamlit Cloud

Technologies: Python, Pandas, NumPy, Scikit-Learn, XGBoost, Streamlit, Joblib
A perfect end-to-end ML deployment project demonstrating production-level model handling.
